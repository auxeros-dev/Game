# Packworm Game

This repository contains a small HTML5 canvas game.

## Playing

Open `index.html` in a modern web browser. For best results, serve it over a local web server, e.g.:

```bash
python3 -m http.server
```

Then visit [http://localhost:8000/](http://localhost:8000/) in your browser.

## Controls
- Arrow keys: move
- Q: Dash (if ability chosen)
- W: Shockwave (if ability chosen)
- E: Phase walls (if ability chosen)
- Enter: Start game / confirm
- Esc: Back to menu
